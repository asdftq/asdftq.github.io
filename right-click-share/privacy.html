Privacy Policy for Right Click Share

Effective date: June 10, 2026

Right Click Share is a Chrome extension that adds a Share item to Chrome's right-click menu. It prepares the page, link, selected text, image, audio, or video chosen by the user for Chrome's Web Share sheet.

Data handled

Right Click Share handles website content only when the user explicitly chooses Share from the context menu. This may include selected text, page URL, link URL, media URL, image bytes, audio bytes, video bytes, file name, MIME type, file size, and page title needed to create the share payload.

Right Click Share does not collect personally identifiable information, health information, financial or payment information, authentication information, personal communications, location, web browsing history, or user activity for tracking.

How data is used

Handled content is used only to create a Web Share payload for the share target selected by the user. For media, the extension may fetch the selected media bytes so the share target receives an actual file instead of only a URL.

Right Click Share does not run analytics, does not profile users, does not sell user data, and does not use user data for advertising, creditworthiness, lending, or any unrelated purpose.

Storage

Most share data is prepared in memory. If Chrome cannot open the in-page share prompt, Right Click Share may temporarily store a pending share payload in chrome.storage.session so the fallback share page can complete the user-requested action. Pending share payloads are removed after expiration when possible and are not synced across devices.

Sharing and transfer

Right Click Share does not send selected content to a developer-operated server. The selected content is provided only to Chrome's Web Share API and then to the share target chosen by the user, such as AirDrop, Messages, Mail, Notes, or another supported app.

The welcome page includes optional newsletter links to Install Worthy on Substack. If the user chooses to open or subscribe through those links, that interaction is handled by Substack and is separate from Right Click Share's local sharing workflow.

Remote code

Right Click Share does not use remote code. The JavaScript, CSS, HTML, and images required by the extension are packaged with the extension.
